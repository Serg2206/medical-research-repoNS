3. **–ß–∏—Ç–∞–µ–º–æ—Å—Ç—å**: –í–º–µ—Å—Ç–æ —Ç–æ–≥–æ, —á—Ç–æ–±—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —É—Å–ª–æ–≤–∏–µ `if i == 0` –¥–ª—è —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è –∑–∞–≥–æ–ª–æ–≤–∫–∞, –º–æ–∂–Ω–æ –ø—Ä–æ—Å—Ç–æ –¥–æ–±–∞–≤–∏—Ç—å –∑–∞–≥–æ–ª–æ–≤–æ–∫ –≤ `valid_lines` –¥–æ —Ü–∏–∫–ª–∞. –≠—Ç–æ —É–ø—Ä–æ—Å—Ç–∏—Ç —Ü–∏–∫–ª –∏ —Å–¥–µ–ª–∞–µ—Ç –∫–æ–¥ –±–æ–ª–µ–µ —á–∏—Ç–∞–µ–º—ã–º.

4. **Best practices**: –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –º–æ–¥—É–ª—è `csv` –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å CSV-—Ñ–∞–π–ª–∞–º–∏ –æ–±–µ—Å–ø–µ—á–∏—Ç –±–æ–ª–µ–µ –Ω–∞–¥–µ–∂–Ω—É—é –æ–±—Ä–∞–±–æ—Ç–∫—É –¥–∞–Ω–Ω—ã—Ö.

–í–æ—Ç —É–ª—É—á—à–µ–Ω–Ω–∞—è –≤–µ—Ä—Å–∏—è —Å–∫—Ä–∏–ø—Ç–∞:

```python
#!/usr/bin/env python3
"""Fix CSV file by removing invalid header rows"""

import csv
import sys

input_file = "training-data/medical_training_data.csv"
output_file = "training-data/cleaned_medical_training_data.csv"

# Read the CSV file and filter out lines that don't have proper CSV structure
# Invalid lines are those that don't contain commas (are just section headers)
valid_lines = []
with open(input_file, "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    for i, line in enumerate(reader):
        if i == 0 or "," in line:  # Keep the header row and lines with CSV data
            valid_lines.append(line)
        else:
            print(f"Removing invalid line {i+1}: {','.join(line)}")

# Write back the cleaned CSV
with open(output_file, "w", encoding="utf-8") as f:
    writer = csv.writer(f)
    writer.writerows(valid_lines)

print(
    f"\nDone! Kept {len(valid_lines)} valid lines, removed {i+1 - len(valid_lines)} invalid lines"
)
```
