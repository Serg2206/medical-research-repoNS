4. **Best Practices**: Рекомендуется использовать модуль `csv` для работы с CSV файлами, так как он обеспечивает большую гибкость и надежность, чем ручной парсинг строк.

Вот улучшенная версия кода:

```python
#!/usr/bin/env python3
"""Fix CSV file by removing invalid header rows"""

import csv
import sys

input_file = "training-data/medical_training_data.csv"
output_file = "training-data/medical_training_data_fixed.csv"

# Open the input and output files
with open(input_file, "r", encoding="utf-8") as inf, \
     open(output_file, "w", encoding="utf-8") as outf:

    # Initialize CSV reader and writer
    reader = csv.reader(inf)
    writer = csv.writer(outf)

    # Initialize counters
    valid_lines = 0
    invalid_lines = 0

    # Process the CSV file
    for i, row in enumerate(reader):
        # Skip if line is just a header without commas
        if i == 0 or len(row) > 1:  # Keep the header row and lines with CSV data
            writer.writerow(row)
            valid_lines += 1
        else:
            print(f"Removing invalid line {i+1}: {','.join(row)}")
            invalid_lines += 1

print(
    f"\nDone! Kept {valid_lines} valid lines, removed {invalid_lines} invalid lines"
)
```

Этот код делает то же самое, что и исходный, но он более эффективен, безопасен и легко читаем.
