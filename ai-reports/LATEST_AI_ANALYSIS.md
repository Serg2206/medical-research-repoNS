4. **Best Practices**: –í–º–µ—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è `readlines()` –∏ `writelines()`, –ª—É—á—à–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏–∏ `csv.reader()` –∏ `csv.writer()`. –≠—Ç–æ —Å–¥–µ–ª–∞–µ—Ç –∫–æ–¥ –±–æ–ª–µ–µ —É—Å—Ç–æ–π—á–∏–≤—ã–º –∫ —Ä–∞–∑–ª–∏—á–Ω—ã–º —Ñ–æ—Ä–º–∞—Ç–∞–º CSV.

–í–æ—Ç –ø—Ä–∏–º–µ—Ä —É–ª—É—á—à–µ–Ω–Ω–æ–≥–æ –∫–æ–¥–∞:

```python
#!/usr/bin/env python3
"""Fix CSV file by removing invalid header rows"""

import csv
import os
import tempfile
import sys

input_file = "training-data/medical_training_data.csv"
temp_file = tempfile.NamedTemporaryFile(delete=False)

# Read the CSV file and write to a temporary file
with open(input_file, "r", encoding="utf-8") as csv_file, open(temp_file.name, "w", encoding="utf-8") as temp:
    reader = csv.reader(csv_file)
    writer = csv.writer(temp)

    for i, row in enumerate(reader):
        if i == 0 or len(row) > 1:  # Keep the header row and rows with CSV data
            writer.writerow(row)
        else:
            print(f"Removing invalid line {i+1}: {','.join(row)}")

# Replace the original file with the cleaned file
os.replace(temp_file.name, input_file)

print(f"\nDone!")
```
