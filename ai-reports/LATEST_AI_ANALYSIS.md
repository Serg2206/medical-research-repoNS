4. **Best practices**: –í–º–µ—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è `readlines()` –∏ `writelines()`, –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–æ–¥—É–ª—å csv –¥–ª—è —á—Ç–µ–Ω–∏—è –∏ –∑–∞–ø–∏—Å–∏ —Ñ–∞–π–ª–æ–≤. –≠—Ç–æ –ø–æ–∑–≤–æ–ª–∏—Ç –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å —Å—Ç—Ä–æ–∫–∏, —Å–æ–¥–µ—Ä–∂–∞—â–∏–µ –∑–∞–ø—è—Ç—ã–µ –≤ –∫–∞–≤—ã—á–∫–∞—Ö.

–í–æ—Ç –ø—Ä–∏–º–µ—Ä —É–ª—É—á—à–µ–Ω–Ω–æ–≥–æ –∫–æ–¥–∞:

```python
#!/usr/bin/env python3
"""Fix CSV file by removing invalid header rows"""

import csv
import sys

input_file = "training-data/medical_training_data.csv"
output_file = "training-data/cleaned_medical_training_data.csv"

# Counter for invalid lines
invalid_lines = 0

with open(input_file, "r", encoding="utf-8") as f_in, open(output_file, "w", encoding="utf-8") as f_out:
    reader = csv.reader(f_in)
    writer = csv.writer(f_out)

    for i, row in enumerate(reader):
        # Keep the header row
        if i == 0:
            writer.writerow(row)
        # Only keep lines with CSV data
        elif len(row) > 1:
            writer.writerow(row)
        else:
            print(f"Removing invalid line {i+1}: {','.join(row)}")
            invalid_lines += 1

print(
    f"\nDone! Kept {i+1-invalid_lines} valid lines, removed {invalid_lines} invalid lines"
)
```
