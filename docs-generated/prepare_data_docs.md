# Документация для prepare_data.py

## Описание

Этот скрипт Python используется для подготовки данных для машинного обучения. Он принимает на вход CSV-файл с данными, разделяет данные на обучающую и тестовую выборки, и сохраняет эти выборки в указанной директории.

## Функции

### `prepare_data(input_csv, output_dir, test_size=0.2)`

#### Описание

Эта функция читает данные из CSV-файла, разделяет их на обучающую и тестовую выборки, и сохраняет эти выборки в указанной директории.

#### Параметры

- `input_csv` (str): Путь к входному CSV-файлу. CSV-файл должен содержать столбцы 'text' и 'label'.
- `output_dir` (str): Директория для сохранения обработанных данных.
- `test_size` (float, optional): Размер тестовой выборки (доля от общего объема данных). По умолчанию равно 0.2.

#### Возвращаемые значения

Функция не возвращает значений, но сохраняет обучающую и тестовую выборки в указанной директории.

## Примеры использования

```python
prepare_data('data.csv', 'output_dir', test_size=0.3)
```

В этом примере данные из файла 'data.csv' будут разделены на обучающую и тестовую выборки, причем тестовая выборка будет составлять 30% от общего объема данных. Обучающая и тестовая выборки будут сохранены в директории 'output_dir' в формате CSV.

## Запуск скрипта из командной строки

```bash
python prepare_data.py --input_csv data.csv --output_dir output_dir --test_size 0.3
```

В этом примере скрипт запускается из командной строки с теми же параметрами, что и в предыдущем примере.
