# Документация для скрипта подготовки данных

## Описание функционала

Этот скрипт предназначен для подготовки данных, которые будут использоваться для обучения и тестирования моделей машинного обучения. Он принимает на вход CSV-файл, содержащий данные, и разделяет эти данные на обучающую и тестовую выборки. Затем он сохраняет эти выборки в указанной директории.

## Функции

### `prepare_data(input_csv, output_dir, test_size=0.2)`

#### Описание

Эта функция читает данные из CSV-файла, разделяет их на обучающую и тестовую выборки, а затем сохраняет эти выборки в указанной директории.

#### Параметры

- `input_csv` (str): Путь к входному CSV-файлу. CSV-файл должен содержать столбцы 'text' и 'label'.
- `output_dir` (str): Директория для сохранения обработанных данных.
- `test_size` (float, optional): Размер тестовой выборки (доля от общего объема данных). По умолчанию равен 0.2.

#### Возвращаемые значения

Функция не возвращает значений. Она сохраняет обработанные данные в указанной директории.

## Примеры использования

```python
prepare_data("data.csv", "output", test_size=0.3)
```

В этом примере данные из файла "data.csv" будут разделены на обучающую и тестовую выборки, причем размер тестовой выборки составит 30% от общего объема данных. Затем обработанные данные будут сохранены в директории "output".
