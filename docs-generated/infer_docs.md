**Дата:** 2025-11-06

# Документация

## Описание функционала

Данный скрипт выполняет инференс (предсказание) с помощью предобученной модели на основе трансформеров для классификации текста. Скрипт использует библиотеки `torch` и `transformers` для загрузки модели и выполнения предсказания. 

## Функции

### infer_model(model_dir, text)

Функция выполняет инференс с помощью модели, загруженной из указанной директории, на входном тексте.

#### Параметры:

- `model_dir` (str): Путь к директории, где сохранена предобученная модель.
- `text` (str): Входной текст, для которого необходимо выполнить предсказание.

#### Возвращаемое значение:

- `probabilities` (list): Список вероятностей для каждого класса, предсказанных моделью.

## Примеры использования

Для использования скрипта необходимо передать два аргумента: путь к модели и текст для анализа. Пример команды для запуска скрипта:

```bash
python script.py --model /path/to/model --text "Текст для анализа"
```

В результате выполнения скрипта будет выведен список вероятностей для каждого класса, предсказанных моделью.

## Примечания

- Для работы скрипта необходимо, чтобы в системе были установлены библиотеки `torch` и `transformers`.
- Модель должна быть совместима с библиотекой `transformers` и поддерживать классификацию текста.
- Текст для анализа должен быть представлен в виде строки.
- Если в системе доступна GPU, скрипт будет автоматически использовать ее для ускорения вычислений. В противном случае будет использоваться CPU.
