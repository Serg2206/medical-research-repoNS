# Документация

## Описание

Этот скрипт Python использует модель классификации последовательностей из библиотеки transformers для анализа и классификации текста. Модель и токенизатор загружаются из предварительно обученной модели, которая указывается в качестве аргумента при запуске скрипта. Текст для анализа также передается в качестве аргумента.

## Функции

### infer_model(model_dir, text)

#### Описание

Эта функция загружает модель и токенизатор из указанной директории, токенизирует входной текст и выполняет инференс модели на токенизированном тексте.

#### Параметры

- `model_dir` (str): Путь к директории, содержащей предварительно обученную модель и токенизатор.
- `text` (str): Текст для анализа.

#### Возвращаемое значение

- `probabilities.tolist()` (list): Список вероятностей для каждого класса, возвращаемый моделью.

## Использование

### Командная строка

Скрипт можно запустить из командной строки, используя следующий формат:

```bash
python script.py --model path_to_model --text "Текст для анализа"
```

### Python

```python
from script import infer_model

model_dir = "path_to_model"
text = "Текст для анализа"
result = infer_model(model_dir, text)
print(f"Результаты предсказания: {result}")
```

В обоих случаях `path_to_model` должен быть заменен на путь к директории с предварительно обученной моделью, а "Текст для анализа" - на текст, который требуется анализировать.
