# Документация

## Описание

Этот код представляет собой скрипт для инференса (предсказания) с использованием предварительно обученной модели классификации последовательностей. Этот скрипт использует библиотеку transformers от Hugging Face для загрузки модели и токенизатора. 

## Функции

### infer_model(model_dir, text)

Эта функция принимает на вход путь к директории модели и текст, который необходимо классифицировать. Она загружает модель и токенизатор из указанной директории, преобразует текст в токены, передает их в модель и возвращает вероятности классов, предсказанные моделью.

#### Параметры

- `model_dir` (str): Путь к директории, содержащей предварительно обученную модель и токенизатор.
- `text` (str): Текст, который необходимо классифицировать.

#### Возвращаемое значение

- `probabilities` (list): Список вероятностей классов, предсказанных моделью.

## Пример использования

```bash
python script.py --model /path/to/model --text "Текст для классификации"
```

В этом примере `/path/to/model` - это путь к директории, содержащей предварительно обученную модель и токенизатор, а `"Текст для классификации"` - это текст, который вы хотите классифицировать. 

После выполнения этой команды вы увидите на экране вероятности классов, предсказанные моделью, в формате списка.
