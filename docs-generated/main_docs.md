# Документация для кода

## Описание функционала

Этот код представляет собой простую нейронную сеть, реализованную с использованием библиотеки PyTorch. Модель предназначена для обучения на наборе данных, который передается в виде параметров. 

## Класс MedicalNet

### Описание

Класс `MedicalNet` представляет собой модель нейронной сети, состоящую из трех полносвязных слоев (`nn.Linear`) и функции dropout для регуляризации.

### Параметры

- `input_size` (int): Размер входного слоя, то есть количество признаков в данных.
- `hidden_size` (int): Размер скрытого слоя, то есть количество нейронов в скрытом слое.
- `output_size` (int): Размер выходного слоя, то есть количество классов для классификации.

### Методы

- `forward(x)`: Прямой проход через модель. Принимает на вход тензор `x` и возвращает тензор, представляющий выход модели.

## Функция train_model

### Описание

Функция `train_model` обучает модель `MedicalNet` на переданных данных.

### Параметры

- `data` (numpy array или torch.Tensor): Данные для обучения модели.
- `labels` (numpy array или torch.Tensor): Метки классов для данных.
- `input_size` (int): Размер входного слоя модели.
- `hidden_size` (int): Размер скрытого слоя модели.
- `output_size` (int): Размер выходного слоя модели.
- `epochs` (int, optional): Количество эпох для обучения. По умолчанию равно 20.
- `batch_size` (int, optional): Размер пакета для обучения. По умолчанию равно 32.

### Возвращаемые значения

Функция не возвращает значений. Она обучает модель на переданных данных.

## Примеры использования

```python
# Создание данных и меток
data = np.random.rand(1000, 10)
labels = np.random.randint(0, 2, 1000)

# Обучение модели
train_model(data, labels, input_size=10, hidden_size=32, output_size=2)
```

В этом примере мы создаем случайные данные и метки, а затем обучаем на них модель. Размер входного слоя равен 10 (по количеству признаков в данных), размер скрытого слоя - 32, а размер выходного слоя - 2 (по количеству классов).
