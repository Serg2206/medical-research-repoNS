# Документация

## Описание функционала

Данный код представляет собой пример простой нейронной сети для обработки медицинских данных. В коде представлены два основных компонента: класс `MedicalNet`, который определяет архитектуру нейронной сети, и функция `train_model`, которая используется для обучения модели.

## Класс MedicalNet

### Описание

Класс `MedicalNet` наследуется от `nn.Module` и представляет собой простую полносвязную нейронную сеть с двумя скрытыми слоями и функцией dropout для предотвращения переобучения.

### Параметры

- `input_size` (int): Размер входного слоя (количество признаков в входных данных).
- `hidden_size` (int): Размер скрытых слоев.
- `output_size` (int): Размер выходного слоя (количество классов для классификации).

### Методы

- `forward(x)`: Пропускает входные данные через нейронную сеть. Возвращает выходные данные после прохождения через все слои.

## Функция train_model

### Описание

Функция `train_model` используется для обучения модели `MedicalNet`. Она создает датасет и загрузчик данных, инициализирует модель и оптимизатор, и затем обучает модель на заданное количество эпох.

### Параметры

- `data` (numpy array): Входные данные для обучения модели.
- `labels` (numpy array): Метки классов для входных данных.
- `input_size` (int): Размер входного слоя (количество признаков в входных данных).
- `hidden_size` (int): Размер скрытых слоев.
- `output_size` (int): Размер выходного слоя (количество классов для классификации).
- `epochs` (int, optional): Количество эпох для обучения модели. По умолчанию равно 20.
- `batch_size` (int, optional): Размер пакета для обучения модели. По умолчанию равно 32.

### Возвращаемые значения

Функция не возвращает значения, но обучает модель `MedicalNet` на входных данных.

## Примеры использования

```python
# Инициализация входных данных и меток
data = np.random.rand(100, 10)
labels = np.random.randint(0, 2, 100)

# Определение размеров слоев
input_size = 10
hidden_size = 5
output_size = 2

# Обучение модели
train_model(data, labels, input_size, hidden_size, output_size)
```
