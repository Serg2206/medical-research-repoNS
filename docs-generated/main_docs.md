# Документация

## Описание функционала

Данный код представляет собой пример простой нейронной сети, реализованной с помощью библиотеки PyTorch. Эта нейронная сеть может быть использована для обучения на различных данных, в данном случае, предполагается использование в медицинских задачах.

## Классы и функции

### Класс `MedicalNet`

Этот класс представляет собой архитектуру нейронной сети. Он наследуется от класса `nn.Module` из библиотеки PyTorch.

#### `__init__(self, input_size, hidden_size, output_size)`

Конструктор класса. Принимает три аргумента: размер входного слоя, размер скрытого слоя и размер выходного слоя. Инициализирует три полносвязных слоя (`nn.Linear`) и слой dropout.

#### `forward(self, x)`

Метод для прямого прохода данных через сеть. Принимает входные данные `x`, пропускает их через слои и возвращает результат.

### Функция `train_model`

Функция для обучения модели. Принимает данные для обучения, метки, размеры слоев и параметры обучения.

#### `train_model(data, labels, input_size, hidden_size, output_size, epochs=20, batch_size=32)`

Принимает следующие аргументы:
- `data`: данные для обучения
- `labels`: метки классов для данных
- `input_size`: размер входного слоя сети
- `hidden_size`: размер скрытого слоя сети
- `output_size`: размер выходного слоя сети
- `epochs`: количество эпох обучения (по умолчанию 20)
- `batch_size`: размер батча для обучения (по умолчанию 32)

Создает датасет и загрузчик данных, инициализирует модель и оптимизатор, и обучает модель на заданных данных.

## Примеры использования

```python
# Создаем данные и метки
data = np.random.rand(100, 10)
labels = np.random.randint(0, 2, 100)

# Задаем размеры слоев
input_size = 10
hidden_size = 50
output_size = 2

# Обучаем модель
train_model(data, labels, input_size, hidden_size, output_size, epochs=100, batch_size=16)
```
