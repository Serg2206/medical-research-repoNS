# Документация

## Описание функционала

Данный код представляет собой простую реализацию нейронной сети с использованием библиотеки PyTorch. Это полносвязная нейронная сеть, которая может быть использована для обучения на любых данных, но в данном случае она предназначена для медицинских данных.

## Класс MedicalNet

### Описание

`MedicalNet` является классом, представляющим архитектуру нейронной сети. Он наследует от `nn.Module`, базового класса для всех нейронных сетей в PyTorch.

### Параметры

- `input_size` (int): Размер входного слоя нейронной сети.
- `hidden_size` (int): Размер скрытого слоя нейронной сети.
- `output_size` (int): Размер выходного слоя нейронной сети.

### Методы

- `__init__(self, input_size, hidden_size, output_size)`: Инициализирует нейронную сеть с заданными размерами слоев.
- `forward(self, x)`: Производит прямой проход через нейронную сеть.

## Функция train_model

### Описание

`train_model` является функцией, которая обучает модель на предоставленных данных.

### Параметры

- `data` (numpy array): Входные данные для обучения модели.
- `labels` (numpy array): Метки классов для входных данных.
- `input_size` (int): Размер входного слоя нейронной сети.
- `hidden_size` (int): Размер скрытого слоя нейронной сети.
- `output_size` (int): Размер выходного слоя нейронной сети.
- `epochs` (int, optional): Количество эпох для обучения модели. По умолчанию равно 20.
- `batch_size` (int, optional): Размер пакета для обучения модели. По умолчанию равно 32.

### Возвращаемые значения

Функция не возвращает значений. Она обучает модель на предоставленных данных.

## Пример использования

```python
# Генерация случайных данных
data = np.random.rand(100, 10)
labels = np.random.randint(0, 2, 100)

# Обучение модели
train_model(data, labels, input_size=10, hidden_size=20, output_size=2)
```
