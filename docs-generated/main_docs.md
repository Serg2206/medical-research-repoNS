# Документация

## Описание функционала

Данный код представляет собой пример простой нейронной сети, реализованной с помощью библиотеки PyTorch. Эта сеть может быть использована для решения задач классификации в медицинских приложениях.

## Класс MedicalNet

### Описание

Класс `MedicalNet` представляет собой архитектуру нейронной сети с тремя полносвязными слоями и промежуточным слоем dropout.

### Параметры

- `input_size` (int): Размер входного слоя (количество входных признаков).
- `hidden_size` (int): Размер скрытых слоев.
- `output_size` (int): Размер выходного слоя (количество классов).

### Методы

- `forward(x)`: Прямой проход по нейронной сети.

## Функция train_model

### Описание

Функция `train_model` используется для обучения модели `MedicalNet`.

### Параметры

- `data` (numpy array или torch tensor): Входные данные для обучения.
- `labels` (numpy array или torch tensor): Метки классов для входных данных.
- `input_size` (int): Размер входного слоя (количество входных признаков).
- `hidden_size` (int): Размер скрытых слоев.
- `output_size` (int): Размер выходного слоя (количество классов).
- `epochs` (int, optional): Количество эпох для обучения. По умолчанию равно 20.
- `batch_size` (int, optional): Размер пакета данных для одной итерации обучения. По умолчанию равно 32.

### Возвращаемые значения

Функция не возвращает значений, но в процессе ее выполнения обучает модель `MedicalNet`.

## Примеры использования

```python
# Создание случайных данных и меток
data = np.random.rand(100, 10)
labels = np.random.randint(0, 2, 100)

# Обучение модели
train_model(data, labels, input_size=10, hidden_size=20, output_size=2)
```
