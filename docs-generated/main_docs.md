# Документация для MedicalNet

## Описание функционала

Данный код представляет собой простую нейронную сеть, которая может быть использована для задач классификации в медицинских приложениях. Она состоит из трех полносвязных слоев и одного слоя dropout для регуляризации.

## Класс MedicalNet

### Описание

Класс `MedicalNet` наследуется от `nn.Module` и представляет собой архитектуру нейронной сети.

### Параметры

- `input_size` (int): Размер входных данных.
- `hidden_size` (int): Размер скрытых слоев.
- `output_size` (int): Размер выходных данных.

### Методы

- `__init__(self, input_size, hidden_size, output_size)`: Инициализирует объект класса.
- `forward(self, x)`: Производит прямой проход через нейронную сеть.

## Функция train_model

### Описание

Функция `train_model` используется для обучения модели `MedicalNet`.

### Параметры

- `data` (numpy.ndarray): Входные данные для обучения.
- `labels` (numpy.ndarray): Метки классов для входных данных.
- `input_size` (int): Размер входных данных.
- `hidden_size` (int): Размер скрытых слоев.
- `output_size` (int): Размер выходных данных.
- `epochs` (int, optional): Количество эпох для обучения. По умолчанию равно 20.
- `batch_size` (int, optional): Размер пакета для обучения. По умолчанию равно 32.

### Возвращаемые значения

Функция не возвращает значения, но обучает модель `MedicalNet`.

## Пример использования

```python
# Импорт необходимых библиотек
import numpy as np

# Генерация случайных данных
np.random.seed(0)
data = np.random.rand(1000, 10)
labels = np.random.randint(0, 2, 1000)

# Обучение модели
train_model(data, labels, input_size=10, hidden_size=32, output_size=2, epochs=50, batch_size=64)
```

Пожалуйста, обратите внимание, что данный код требует установленной библиотеки PyTorch.
