# Документация для кода

## Описание функционала

Данный код представляет собой пример простой нейронной сети, созданной с помощью фреймворка PyTorch. Эта нейронная сеть предназначена для обучения на наборе данных, представляющих собой медицинскую информацию.

## Описание классов и функций

### Класс `MedicalNet`

#### Описание

Класс `MedicalNet` представляет собой модель нейронной сети, состоящей из двух полносвязных слоев и одного выходного слоя. В процессе обучения используется функция активации ReLU и dropout для предотвращения переобучения.

#### Параметры

- `input_size` (int): Размер входного слоя.
- `hidden_size` (int): Размер скрытого слоя.
- `output_size` (int): Размер выходного слоя.

#### Методы

- `forward(x)`: Производит прямой проход данных через нейронную сеть.

### Функция `train_model`

#### Описание

Функция `train_model` используется для обучения модели `MedicalNet`.

#### Параметры

- `data` (numpy.ndarray): Массив данных для обучения.
- `labels` (numpy.ndarray): Массив меток для данных.
- `input_size` (int): Размер входного слоя.
- `hidden_size` (int): Размер скрытого слоя.
- `output_size` (int): Размер выходного слоя.
- `epochs` (int, optional): Количество эпох для обучения. По умолчанию равно 20.
- `batch_size` (int, optional): Размер пакета данных для обучения. По умолчанию равно 32.

#### Возвращаемые значения

Функция не возвращает значений.

## Примеры использования

```python
# Инициализация данных
data = np.random.rand(1000, 10)
labels = np.random.randint(0, 2, 1000)

# Обучение модели
train_model(data, labels, input_size=10, hidden_size=50, output_size=2, epochs=100, batch_size=64)
```
