# Документация

## Описание функционала

Данный код представляет собой пример реализации нейронной сети для медицинских задач с использованием библиотеки PyTorch. В коде определены следующие функции:
- `MedicalNet`: класс, определяющий архитектуру нейронной сети.
- `train_model`: функция для обучения модели на предоставленных данных.

## Класс `MedicalNet`

### Описание

Класс `MedicalNet` представляет собой нейронную сеть, состоящую из двух полносвязных слоев и одного выходного слоя. В классе используется dropout для предотвращения переобучения.

### Параметры

- `input_size`: размер входного слоя (тип: int).
- `hidden_size`: размер скрытого слоя (тип: int).
- `output_size`: размер выходного слоя (тип: int).

### Методы

- `forward(x)`: прямой проход через нейронную сеть.

## Функция `train_model`

### Описание

Функция `train_model` используется для обучения модели `MedicalNet` на предоставленных данных.

### Параметры

- `data`: входные данные для обучения (тип: numpy array).
- `labels`: метки классов для входных данных (тип: numpy array).
- `input_size`: размер входного слоя (тип: int).
- `hidden_size`: размер скрытого слоя (тип: int).
- `output_size`: размер выходного слоя (тип: int).
- `epochs`: количество эпох обучения (тип: int, по умолчанию 20).
- `batch_size`: размер батча для обучения (тип: int, по умолчанию 32).

### Возвращаемые значения

Функция не возвращает значений, но обучает модель `MedicalNet` на предоставленных данных.

## Примеры использования

```python
# Инициализация данных
data = np.random.rand(1000, 10)
labels = np.random.randint(0, 2, 1000)

# Определение размеров слоев
input_size = 10
hidden_size = 50
output_size = 2

# Обучение модели
train_model(data, labels, input_size, hidden_size, output_size, epochs=100, batch_size=64)
```
