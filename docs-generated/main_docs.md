# Документация

## Описание функционала

Этот код представляет собой простую нейронную сеть, которая может быть использована для обучения модели на основе медицинских данных. Модель состоит из трех полносвязных слоев (fc1, fc2, fc3) и слоя dropout, который помогает предотвратить переобучение.

## Описание классов и функций

### Класс `MedicalNet`

Этот класс представляет архитектуру нейронной сети.

#### Параметры конструктора

- `input_size` (int): Размер входного слоя.
- `hidden_size` (int): Размер скрытого слоя.
- `output_size` (int): Размер выходного слоя.

#### Методы

- `forward(x)`: Пропускает входные данные через нейронную сеть.

### Функция `train_model`

Эта функция обучает модель на основе данных и меток.

#### Параметры

- `data` (numpy.ndarray): Входные данные для обучения.
- `labels` (numpy.ndarray): Метки для входных данных.
- `input_size` (int): Размер входного слоя.
- `hidden_size` (int): Размер скрытого слоя.
- `output_size` (int): Размер выходного слоя.
- `epochs` (int, optional): Количество эпох для обучения. По умолчанию равно 20.
- `batch_size` (int, optional): Размер пакета для обучения. По умолчанию равно 32.

#### Возвращаемые значения

Функция не возвращает значений, но обучает модель на основе входных данных.

## Примеры использования

```python
# Импорт необходимых библиотек
import numpy as np

# Генерация случайных данных и меток
data = np.random.rand(1000, 10)
labels = np.random.randint(0, 2, 1000)

# Обучение модели
train_model(data, labels, input_size=10, hidden_size=32, output_size=2)
```

В этом примере мы генерируем 1000 случайных примеров с 10 признаками каждый и обучаем модель на этих данных. Метки представляют собой случайные значения 0 или 1, что подразумевает бинарную классификацию.
