# Документация

## Описание функционала

Этот код представляет собой простую нейронную сеть, которая используется для обучения на медицинских данных. Сеть состоит из трех полносвязных слоев и одного слоя dropout. Обучение модели осуществляется с использованием функции потерь CrossEntropyLoss и оптимизатора Adam.

## Класс MedicalNet

### Описание

Класс `MedicalNet` представляет собой архитектуру нейронной сети. Он наследуется от класса `nn.Module` из библиотеки PyTorch.

### Параметры

- `input_size` (int): Размер входного слоя нейронной сети.
- `hidden_size` (int): Размер скрытого слоя нейронной сети.
- `output_size` (int): Размер выходного слоя нейронной сети.

### Методы

- `__init__(self, input_size, hidden_size, output_size)`: Инициализирует объект класса `MedicalNet`.
- `forward(self, x)`: Производит прямой проход через нейронную сеть.

## Функция train_model

### Описание

Функция `train_model` используется для обучения модели `MedicalNet`.

### Параметры

- `data` (numpy array): Массив данных для обучения.
- `labels` (numpy array): Массив меток для обучения.
- `input_size` (int): Размер входного слоя нейронной сети.
- `hidden_size` (int): Размер скрытого слоя нейронной сети.
- `output_size` (int): Размер выходного слоя нейронной сети.
- `epochs` (int, optional): Количество эпох для обучения. По умолчанию равно 20.
- `batch_size` (int, optional): Размер батча для обучения. По умолчанию равно 32.

### Возвращаемые значения

Функция не возвращает значений. Она обучает модель `MedicalNet` на предоставленных данных.

## Примеры использования

```python
# Импорт необходимых библиотек
import numpy as np

# Генерация случайных данных и меток
data = np.random.rand(1000, 10)
labels = np.random.randint(0, 2, 1000)

# Определение размеров слоев
input_size = 10
hidden_size = 50
output_size = 2

# Обучение модели
train_model(data, labels, input_size, hidden_size, output_size)
```
