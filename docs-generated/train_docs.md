# Документация для кода

## Описание функционала

Этот код предназначен для обучения модели классификации текста на основе BERT. Он включает в себя функции для подготовки данных, создания датасета и обучения модели.

## Классы и функции

### Класс `MedicalDataset`

Этот класс представляет собой настраиваемый датасет, который используется для загрузки и предобработки данных.

#### Параметры
- `data`: DataFrame, содержащий данные для обучения. Ожидается, что он будет содержать столбцы "text" и "label".
- `tokenizer`: Токенизатор из библиотеки transformers, который используется для преобразования текста в токены.
- `max_length`: Максимальная длина последовательности токенов. По умолчанию равна 128.

#### Методы
- `__len__`: Возвращает количество элементов в датасете.
- `__getitem__`: Возвращает токенизированный текст и метку для указанного индекса.

### Функция `prepare_data`

Эта функция используется для подготовки данных: она загружает данные из CSV-файла, разделяет их на обучающую и тестовую выборки и сохраняет результаты.

#### Параметры
- `input_csv`: Путь к входному CSV-файлу.
- `output_dir`: Директория для сохранения обработанных данных.
- `test_size`: Размер тестовой выборки в диапазоне от 0 до 1. По умолчанию равен 0.2.

## Примеры использования

```python
# Подготовка данных
prepare_data("data.csv", "output_dir")

# Создание датасета
tokenizer = AutoTokenizer.from_pretrained(MODEL_NAME)
data = pd.read_csv("output_dir/train.csv")
dataset = MedicalDataset(data, tokenizer)

# Получение элемента датасета
item = dataset[0]
print(item)
```
