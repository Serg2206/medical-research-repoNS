# Документация для кода

## Описание функционала

Этот код предназначен для обучения модели классификации последовательностей на основе BERT. Он включает в себя загрузку данных, их подготовку, создание датасета и обучение модели. 

## Классы и функции

### Класс MedicalDataset

Этот класс представляет собой датасет, который используется для обучения и тестирования модели. 

#### `__init__(self, data, tokenizer, max_length=128)`

Инициализирует экземпляр класса.

- `data`: DataFrame, содержащий данные для обучения или тестирования. Должен содержать столбцы "text" и "label".
- `tokenizer`: Токенизатор из библиотеки transformers, который будет использоваться для преобразования текста в токены.
- `max_length`: Максимальная длина последовательности. По умолчанию равна 128.

#### `__len__(self)`

Возвращает количество элементов в датасете.

#### `__getitem__(self, idx)`

Возвращает токенизированный текст и метку для элемента с индексом `idx`.

### Функция prepare_data

Эта функция предназначена для подготовки данных: она разделяет входные данные на обучающую и тестовую выборки.

#### `prepare_data(input_csv, output_dir, test_size=0.2)`

- `input_csv`: Путь к входному CSV-файлу.
- `output_dir`: Директория для сохранения обработанных данных.
- `test_size`: Доля данных, которая будет использоваться для тестирования. По умолчанию равна 0.2.

## Примеры использования

```python
# Подготовка данных
prepare_data("input.csv", "output_dir")

# Создание датасета
tokenizer = AutoTokenizer.from_pretrained(MODEL_NAME)
data = pd.read_csv("output_dir/train.csv")
dataset = MedicalDataset(data, tokenizer)

# Получение элемента датасета
item = dataset[0]
```

Обратите внимание, что этот код требует наличия библиотек `pandas`, `torch`, `sklearn` и `transformers`.
