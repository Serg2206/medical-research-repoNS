## 🔗 Интеграция с существующими workflow

### docs-improver.yml (РАСШИРЕН)

```yaml
Триггеры:
  - Расписание: Среда 5:00 UTC
  - Push: *.py, *.md, monograph/**  ← ДОБАВЛЕНО
  - Ручной запуск

Jobs:
  1. improve-docs (существующий)
     - Документация Python-файлов
  
  2. improve-monograph (НОВЫЙ)
     - AI-улучшение глав монографии
     - Генерация иллюстраций
     - Проверка качества
```

### monograph-generator.yml (НОВЫЙ)

```yaml
Триггеры:
  - Ручной запуск
  - Push: monograph/**
  - Расписание: Воскресенье 6:00 UTC

Jobs:
  1. generate-monograph
     - Сборка полной монографии
     - Экспорт в LaTeX/PDF
     - Валидация контента
  
  2. create-illustrations
     - Генерация медицинских схем
     - Обработка изображений
     - Добавление аннотаций
  
  3. analyze-clinical-data
     - Статистический анализ
     - Создание графиков
     - Генерация таблиц
```
