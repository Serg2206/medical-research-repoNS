# Visual Surgical Content Collection

## Quick Start

This directory contains **27 high-quality images** from open-access surgical publications.

### What's Inside

- **`surgical-photos/`** - Intraoperative Surgical Photographs (6 images)
- **`anatomical-diagrams/`** - Anatomical Diagrams and Illustrations (5 images)
- **`procedure-steps/`** - Step-by-Step Procedure Diagrams (4 images)
- **`3d-reconstructions/`** - 3D Reconstructions and Advanced Imaging (12 images)
